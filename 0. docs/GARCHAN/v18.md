[Опыты](../Диссертация/I%20ГЛАВА/1.%20ВВЕДЕНИЕ/1.1.3%20ЭКСПЕРИМЕНТЫ.md) указывают на то что, энтропия может менятся на характерном размере.

Предлагаемый прототип использует **материалы с управляемыми свойствами** и **физические эффекты**, позволяющие генерировать градиенты плотности/напряжённости полей **без внешнего вакуумирования** — исключительно за счёт протекания тока или внешних стимулов.


### Ключевые принципы работы

1. **Электрострикция и пьезоэффект** — локальное изменение объёма/формы материала под действием электрического поля.  
2. **Фазовые переходы с изменением плотности** — переключение между состояниями вещества (твёрдое ↔ жидкое) при нагреве током.  
3. **Ионная миграция в твёрдых электролитах** — перераспределение массы за счёт движения ионов.  
4. **Магнитострикция** — изменение размеров ферромагнитных материалов в магнитном поле (создаётся током).  


### Состав прототипа

#### 1. Активный элемент — многослойный композит
- **Внешний слой**: пьезоэлектрик (PZT — цирконат‑титанат свинца) или диэлектрический эластомер.  
  *Функция*: деформация под напряжением → локальное изменение плотности.  
- **Средний слой**: материал с фазовым переходом (например, сплав Вуда, температура плавления ~60 °C).  
  *Функция*: при пропускании тока плавится → резкое изменение плотности.  
- **Внутренний слой**: твёрдый электролит (например, стабилизированный оксидом иттрия диоксид циркония, YSZ).  
  *Функция*: ионная проводимость → перераспределение массы при токе.  
- **Ферромагнитные включения** (частицы железа или феррита) в слоях.  
  *Функция*: магнитострикционный отклик на магнитное поле от тока.  


#### 2. Электромагнитная система
- **Кольцевые катушки** вокруг композита (медь, 100–200 витков).  
  *Функция*: создание переменного магнитного поля при пропускании импульсного тока.  
- **Токоподводы** из серебра или меди с низким сопротивлением.  


#### 3. Датчики (встроены в слои или на поверхности)
- **Пьезодатчики** — фиксация акустических аномалий.  
- **Термопары** (тип K) — контроль локального нагрева.  
- **Датчики магнитного поля** (Hall‑сенсоры) — измерение индукции.  
- **Ёмкостные датчики** — регистрация изменения диэлектрической проницаемости (связано с плотностью).  
- **Оптические волокна с брэгговскими решётками** — измерение деформации/напряжения в реальном времени.  


#### 4. Управляющая электроника
- **Микроконтроллер** (STM32 или ESP32) с АЦП и ЦАП.  
- **Импульсный источник тока** (до 10 А, длительность импульса 1–10 мс).  
- **Генератор переменного поля** (частота 1–10 кГц).  
- **SD‑карта/флеш‑память** для записи данных.  
- **Интерфейс** (USB, Bluetooth) для передачи данных.  


#### 5. Корпус и термостабилизация
- **Герметичный корпус** из немагнитного сплава (алюминий, титан).  
- **Теплоотвод** (медные пластины + вентилятор) — предотвращение перегрева.  
- **Термоизоляционные прокладки** (аэрогель) — локализация тепловых эффектов.  



### Принцип работы


1. **Инициализация**:  
   - Композит находится в исходном состоянии (твёрдый сплав, неподвижные ионы).  
   - Датчики калибруются на фоновые значения.  

2. **Фаза активации**:  
   - На катушки подаётся импульсный ток (1–10 А, 1–10 мс) → создаётся магнитное поле.  
   - В ферромагнитных включениях возникает магнитострикция → локальная деформация.  
   - Одновременно через электролит пропускается постоянный ток → ионы мигрируют → изменение локальной плотности.  
   - При необходимости подаётся дополнительный ток на нагрев → плавление сплава Вуда → скачкообразное изменение плотности.  

3. **Регистрация сигналов**:  
   - Пьезодатчики фиксируют акустические волны от деформации.  
   - Ёмкостные датчики отмечают изменение диэлектрической проницаемости.  
   - Термопары регистрируют локальный нагрев (джоулево тепло + фазовый переход).  
   - Hall‑сенсоры отслеживают динамику магнитного поля.  
   - Оптические волокна фиксируют механическую деформацию.  
4. **Циклирование**:  
   - Повторяйте импульсы с разной амплитудой/частотой для поиска резонансных режимов.  
   - Чередуйте нагрев и охлаждение для многократного фазового перехода.  


### Преимущества прототипа

- **Автономность**: не требует вакуумного оборудования — все эффекты создаются током.  
- **Многопараметричность**: одновременно измеряются акустика, тепло, магнитное поле, деформация.  
- **Масштабируемость**: можно увеличивать число слоёв или площадь композита.  
- **Управляемость**: параметры тока (амплитуда, частота, длительность) позволяют точно настраивать воздействие.  
- **Ремонтопригодность**: модульная конструкция допускает замену отдельных слоёв.  


### Ограничения и пути их преодоления

- **Перегрев**: используйте импульсные режимы и теплоотводы.  
- **Деградация материалов**: выбирайте термостабильные пьезоэлектрики (PZT) и электролиты (YSZ).  
- **Шумы датчиков**: экранируйте провода, применяйте цифровые фильтры в ПО.  
- **Нелинейность откликов**: калибруйте систему на эталонных образцах.  


### ПО для обработки данных (пример на Python)
```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.signal import find_peaks


# Загрузка данных с датчиков
data = np.loadtxt('sensor_data.csv', delimiter=',')

time, acoustic, temp, field, strain = data.T



# Поиск аномалий
peaks, _ = find_peaks(acoustic, height=0.5)  # Акустические всплески
temp_anomalies = np.where(np.diff(temp) > 0.1)[0]  # Резкие скачки температуры


# Визуализация
plt.figure(figsize=(12, 8))
plt.subplot(4, 1, 1); plt.plot(time, acoustic); plt.title('Акустика')
plt.subplot(4, 1, 2); plt.plot(time, temp); plt.title('Температура')
plt.subplot(4, 1, 3); plt.plot(time, field); plt.title('Магнитное поле')
plt.subplot(4, 1, 4); plt.plot(time, strain); plt.title('Деформация')
plt.show()
```


### Этапы реализации
1. **Изготовление композита**:  
   - Нанесите слои PZT, сплава Вуда и YSZ методом трафаретной печати или ламинирования.  
   - Внедрите датчики в межслойные пространства.  
2. **Сборка корпуса** с теплоотводом и экранированием.  
3. **Калибровка датчиков** на эталонных образцах.  
4. **Тестовые импульсы** с постепенным увеличением тока.  
5. **Анализ данных** на поиск корреляций между сигналами.  


**Итог**: прототип позволяет:  
- создавать градиенты плотности без вакуума;  
- регистрировать мультифизические аномалии;  
- исследовать «энергию X» в управляемых условиях.
