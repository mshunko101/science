### Что мы проделали


1. **Сформулировали условие согласованности**  
Исходное равенство двух выражений для $`\alpha`$ при $`n=3`$ и $`L=2R`$ привело к условию:  

$$
\rho R^3 = \frac{3m}{4\pi}
$$

Это математически эквивалентно формуле массы однородного шара $`m = \rho \cdot \frac{4}{3}\pi R^3`$

3. **Проверили условие для Земли**  
- Подставили реальные данные (масса, радиус, плотность).  
- Обнаружили, что левая и правая части совпадают с погрешностью < 0,1 %.  
- Сделали вывод: для однородного шара условие выполняется автоматически.


4. **Распространили анализ на все планеты Солнечной системы**  
- Собрали данные по массе, радиусу и плотности для 8 планет.  
- Для каждой вычислили:  
  - левую часть $`\rho R^3`$ ;  
  - правую часть $`\frac{3m}{4\pi}`$ ;  
  - относительную погрешность $`\delta`$ .  
- Составили таблицу результатов.

5. **Проанализировали отклонения**  
- Выделили группы планет по величине $`\delta`$ :  
  - $`\delta \approx 0`$ % (Венера, Земля);  
  - $`\delta < 1`$ % (Марс, Юпитер, Уран, Нептун);  
  - $`\delta \sim 2`$ % (Сатурн);  
  - $`\delta > 50`$ % (Меркурий).  
- Предположили причины отклонений:  
  - неоднородность внутреннего строения (Меркурий);  
  - сложная структура газовых гигантов (Сатурн).


6. **Интерпретировали физический смысл**  
- Условие согласованности — это проверка модели «однородный шар».  
- Если оно выполняется, то $`X \neq 0`$ — физически осмысленное решение.  
- Если нарушается — нужно уточнять модель (геометрию, распределение плотности).
