## Сборка установки для поглощения и изучения энергии X


Ниже — пошаговая инструкция по созданию **рабочей установки**, способной:  
- создавать градиенты плотности (вакуум/избыточное давление);  
- фиксировать потенциальные проявления энергии X;  
- исключать артефакты.

---

### 1. Компоненты и материалы

**Основа конструкции**:
- Стеклянная колба объёмом 1 л с притёртой крышкой (диаметр горловины ≥ 5 см).  
- Силиконовые трубки (внутренний диаметр 6 мм, длина 50 см).  
- Игольчатый клапан (для регулировки потока газа).  
- Мини‑насос 12 В (производительность ≥ 20 л/мин, вакуум до 0,05 бар).  

**Датчики давления**:
- Манометр 1 (вход): 0–1 бар, точность ± 0,01 бар.  
- Манометр 2 (внутри камеры): аналогичный.  

**Температурный мониторинг**:
- 3 термопары типа K (хромель‑алюмель), диапазон −50… +800 °C.  
- Пирометр MLX90614 (бесконтактный ИК‑датчик, точность ± 0,5 °C).  

**Регистрация аномалий**:
- Пьезодатчик + усилитель (диапазон 20 Гц–20 кГц).  
- SDR‑приёмник RTL‑SDR с рупорной антенной (диапазон 100 кГц–1,7 ГГц).  
- Камера Raspberry Pi Camera v2 с ИК/УФ‑фильтрами.  
- Датчик электрического поля (на базе полевого транзистора).  


**Управление и регистрация**:
- Контроллер: Raspberry Pi 4 (или Arduino Mega 2560 + shield для аналоговых входов).  
- SD‑карта 32 ГБ для хранения данных.  
- Блок питания 12 В, 2 А.  


**Герметизация и крепление**:
- Силиконовый герметик.  
- Вакуумная смазка.  
- Металлические хомуты/стяжки.  
- Перфорированная монтажная панель (для фиксации датчиков).  


---

### 2. Пошаговая сборка


#### Шаг 1. Подготовка камеры
1. Проделайте в крышке колбы **5 отверстий** (диаметр под датчики):  
   - для манометра 1 (входное давление);  
   - для трубки манометра 2 (вывод внутрь камеры);  
   - для трёх термопар;  
   - для пьезодатчика;  
   - для силиконовой трубки насоса.  
2. Обработайте края отверстий наждачной бумагой (удаление заусенцев).  
3. Установите датчики в отверстия, зафиксируйте **силиконовым герметиком**.  
4. Нанесите **вакуумную смазку** на резьбу крышки и уплотнительное кольцо.  
5. Закрутите крышку, проверьте герметичность (см. Шаг 3).  


#### Шаг 2. Монтаж системы вакуумирования
1. Подключите насос к **игольчатому клапану** через силиконовую трубку.  
2. Соедините клапан с **входным штуцером камеры** (отверстие для насоса).  
3. Откройте клапан, включите насос, откачайте воздух до давления 0,2 бар.  
4. Закройте клапан, отключите насос.  
5. Наблюдайте за давлением **5 минут**: падение должно быть < 0,02 бар. Если больше — ищите утечки (см. раздел «Контроль герметичности»).  


#### Шаг 3. Подключение датчиков температуры
1. Разместите термопары:  
   - **Т1**: у верхней стенки камеры;  
   - **Т2**: в центре объёма;  
   - **Т3**: у дна камеры.  
2. Закрепите провода хомутами, чтобы избежать смещения.  
3. Подключите термопары к **аналоговым входам контроллера** (через усилитель, если требуется).  
4. Калибруйте: погрузите термопары в воду с известной температурой (25 °C, 50 °C), сравните с показаниями контроллера.  

#### Шаг 4. Установка пирометра
1. Закрепите MLX90614 на стойке напротив **центра камеры**.  
2. Направьте объектив строго в геометрический центр объёма.  
3. Подключите к контроллеру через **I²C‑интерфейс** (Raspberry Pi) или аналоговый выход (Arduino).  
4. Проверьте показания: при комнатной температуре сигнал должен быть стабильным.  

#### Шаг 5. Монтаж акустического датчика
1. Прикрепите пьезодатчик к **внешней стенке камеры** (лучше в средней зоне).  
2. Подключите к усилителю, затем к **аналоговому входу контроллера**.  
3. Протестируйте: постучите по камере — должен быть всплеск сигнала.  
#### Шаг 6. Установка SDR‑системы
1. Разместите **рупорную антенну** на расстоянии 10 см от камеры.  
2. Подключите SDR‑приёмник к **USB‑порту Raspberry Pi**.  
3. Запустите ПО (например, SDR# или Gqrx) для мониторинга спектра.  
4. Запишите фоновый ЭМ‑фон (без вакуума).  
#### Шаг 7. Монтаж камеры
1. Установите Raspberry Pi Camera v2 напротив **прозрачного участка камеры**.  
2. Закрепите ИК/УФ‑фильтры перед объективом (по очереди для разных тестов).  
3. Подключите камеру к **CSI‑порту Raspberry Pi**.  
4. Проверьте изображение: в кадре должен быть весь объём камеры.  
#### Шаг 8. Датчик электрического поля
1. Расположите датчик у боковой стенки камеры (расстояние 2–3 см).  
2. Подключите к аналоговому входу контроллера.  
3. Запишите фоновые значения (без вакуума/давления).  
#### Шаг 9. Сборка электрической схемы
1. Соедините все датчики с контроллером согласно распиновке.  
2. Подключите блок питания к насосу и контроллеру.  
3. Установите SD‑карту в Raspberry Pi/Arduino.  
4. Проверьте питание всех узлов (индикаторы, отсутствие перегрева).

### 3. Контроль герметичности


1. **Мыльный раствор**:  
   - нанесите на все соединения (крышка, штуцеры, трубки).  
   - при утечке появятся пузырьки.  
2. **Вакуумный тест**:  
   - откачайте воздух до 0,3 бар;  
   - закройте клапан, засеките время падения давления на 0,05 бар.  
   - норма: > 10 минут.  
3. **Тепловой тест**:  
   - нагрейте камеру феном (T +40 °C);  
   - утечка проявится как локальное охлаждение.  

Если обнаружены утечки:  
- зачистите поверхность;  
- нанесите свежий силикон/смазку;  
- повторите тест.  

---

### 4. Предварительная калибровка

1. Оставьте установку в режиме **«без вакуума»** на 1 час.  
2. Запишите **фоновые значения** для всех датчиков (каждые 10 сек).  
3. Рассчитайте **среднее** и **σ** (стандартное отклонение) для каждого канала.  
4. Убедитесь, что колебания < 5 % от среднего.  

**Примеры фоновых значений**:  
- Температура (Т1–Т3): 22 ± 0,5 °C;  
- ИК‑сигнал (MLX90614): 21,8 ± 0,3 °C;  
- Акустика (пьезодатчик): 0,1 ± 0,02 В;  
- ЭМ‑спектр (SDR): −80 ± 5 дБм;  
- Электрическое поле: 0,05 ± 0,01 В.

### 5. Инструменты и ПО

**ПО**:

- VLC/OpenCV: для записи видео с камеры.

1. **Вакуум**:  
   - не откачивайте ниже 0,05 бар в стеклянных камерах (риск разрушения);  
   - работайте в защитных очках.  
2. **Электрика**:  
   - изолируйте все соединения термостойким силиконом;  
   - используйте блоки питания с гальванической развяз
  
## 7. Настройка программного обеспечения


### 7.1. Для Raspberry Pi (основная конфигурация)

1. **Установка ОС и библиотек**:
   ```bash
   sudo apt update
   sudo apt install python3-pip python3-numpy python3-matplotlib python3-smbus
   pip3 install adafruit-circuitpython-mlx90614
   ```

2. **Проверка датчиков**:
   - Убедитесь, что MLX90614 виден по I²C:  
     ```bash
     i2cdetect -y 1
     ```
     Должно отобразиться устройство с адресом `0x5A`.
   - Проверьте работу термопар через АЦП (ADS1115).

3. **Запуск скрипта сбора данных** (пример `collect_data.py`):
   ```python
   import time
   import board
   import busio
   import adafruit_mlx90614
   from adafruit_ads1x15.ads1115 import ADS1115
   import csv
   import numpy as np

   # Инициализация интерфейсов
   i2c = busio.I2C(board.SCL, board.SDA)
   mlx = adafruit_mlx90614.MLX90614(i2c)
   ads = ADS1115(i2c)


   # Открытие файла для записи
   with open('experiment_data.csv', 'w', newline='') as file:
       writer = csv.writer(file)
       writer.writerow(['Timestamp', 'IR_Temp', 'T1', 'T2', 'T3', 
                        'Pressure_In', 'Pressure_Cam', 'Acoustic', 'Electric_Field'])


   # Основной цикл сбора данных
   while True:
       # Считывание показаний
       ir_temp = mlx.object_temperature
       t1 = ads.read_adc(0, gain=1) * 0.00488  # Термопара 1
       t2 = ads.read_adc(1, gain=1) * 0.00488  # Термопара 2
       t3 = ads.read_adc(2, gain=1) * 0.00488  # Термопара 3
       p_in = ads.read_adc(3, gain=1) * 0.00488 * 10  # Манометр 1
       p_cam = ads.read_adc(4, gain=1) * 0.00488 * 10  # Манометр 2
       acous = ads.read_adc(5, gain=1) * 0.00488  # Пьезодатчик
       e_field = ads.read_adc(6, gain=1) * 0.00488  # Датчик поля


       # Запись в CSV
       with open('experiment_data.csv', 'a', newline='') as file:
           writer = csv.writer(file)
           writer.writerow([time.time(), ir_temp, t1, t2, t3,
                           p_in, p_cam, acous, e_field])


       time.sleep(1)  # Интервал записи — 1 секунда
   ```

4. **Автозапуск скрипта** (добавление в `crontab`):  
   ```bash
   crontab -e
   # Добавить строку:
   @reboot /usr/bin/python3 /home/pi/collect_data.py &
   ```

### 7.2. Для Arduino (альтернативная конфигурация)

1. Подключите датчики к аналоговым входам A0–A7.  
2. Загрузите скетч через Arduino IDE (используйте библиотеку `SD.h` для записи на карту памяти).  
3. Пример кода:  
   ```cpp
   #include <SD.h>

   File dataFile;

   void setup() {
       Serial.begin(9600);
       SD.begin(4);  // Пин CS для SD-модуля
       dataFile = SD.open("data.txt", FILE_WRITE);
       if (dataFile) {
           dataFile.println("Time,IR_Temp,T1,T2,T3,P_In,P_Cam,Acous,E_Field");
           dataFile.close();
       }
   }

   void loop() {
       // Чтение датчиков (аналогично Python-коду)
       // Запись в файл
       delay(1000);
   }
   ```

3. **Проведите «холостой» тест** (без вакуума/давления):

## 9. Оптимизация конструкции


### 9.1. Улучшение герметичности
- Замените стеклянные трубки на **металлические капилляры** с вакуумными уплотнениями.  
- Используйте **фторопластовые прокладки** вместо резиновых.  


### 9.2. Повышение чувствительности датчиков
- Для термопар: добавьте **экранированные кабели** (защита от наводок).  
- Для пьезодатчика: установите **резонансную пластину** (усиление акустических колебаний).  
- Для SDR: используйте **направленную антенну** с коэффициентом усиления ≥ 10 дБ.  

### 9.3. Автоматизация вакуума
1. Подключите **реле** к насосу.  
2. Напишите скрипт на Python для поддержания заданного давления:  
   ```python
   target_pressure = 0.3  # бар
   if p_cam < target_pressure - 0.02:
       GPIO.output(pump_pin, GPIO.HIGH)  # Включить насос
   elif p_cam > target_pressure + 0.02:
       GPIO.output(pump_pin, GPIO.LOW)   # Выключить насос
   ```

## 10. Техника безопасности (дополнение)

1. **При работе с вакуумом**:  
   - Не оставляйте установку без присмотра более чем на 1 час.  
   - Установите **предохранительный клапан** (срабатывание при P > 1,5 бар).  
2. **Электробезопасность**:  
   - Все соединения должны быть **изолированы термостойким силиконом**.  
   - Используйте **УЗО** (устройство защитного отключения) в цепи питания.  
3. **Термические риски**:  
   - При нагреве камеры выше 60 °C используйте **перчатки**.  
   - Обеспечьте **принудительную вентиляцию** (кулер на выходе из камеры).

## 11. Чек‑лист готовности установки


Перед началом экспериментов проверьте:
1. Герметичность камеры (мыльный тест).  
2. Калибровку всех датчиков (сравнение с эталонными приборами).  
3. Запись данных на SD‑карту (наличие файла `experiment_data.csv`).  
4. Стабильность питания (напряжение на блоках 12 В ± 0,2 В).  
5. Отсутствие наводок (фоновые значения в норме).  

6. Работу насоса (достижение P = 0,3 бар за 2 минуты).  
7. Корректность временных меток в логах.
